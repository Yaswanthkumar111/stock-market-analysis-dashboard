# Stock Market Analysis Dashboard

## Overview
#### This project focuses on analyzing historical stock market data to understand price trends, volatility, and trading behavior. The goal was to build a complete data analytics pipeline starting from raw data processing to interactive dashboard creation.

## Objective
#### The main objective of this project is to:
* Analyze stock price movements over time
* Identify patterns using moving averages
* Understand volatility and daily returns
* Build an interactive dashboard for better insights

## Tools & Technologies
* Python – Data cleaning and feature engineering
* MySQL – Data storage and querying
* Power BI – Data visualization
* CSV/Excel – Data source

## Project Workflow
### 1.Data Collection
  The dataset was collected in CSV format containing stock price details.
### 2.Data Processing
  Using Python, additional columns like:
  * Daily Return
  * Log Return
  * Moving Averages
  * Volatility were created.
### 3.Database Integration
  The processed data was stored in MySQL using structured tables.
### 4.Data Visualization
  Power BI was used to build an interactive dashboard.

## Dashboard Features
* KPI Metrics (Price, Return, Volume)
* Stock Price Trend Analysis
* Moving Average Comparison (MA20 vs MA50)
* Daily Return Visualization
* Trading Volume Analysis
* Volatility Tracking

## Key Insights
* The stock shows a general upward trend over time.
* Short-term fluctuations are visible in daily returns
* Moving averages help identify trend direction
* Volatility indicates risk levels in the market
* Trading volume reflects investor activity
